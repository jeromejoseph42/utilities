# utilities
Simple scripts for adding these functionalities:
1. Paste words 1-by-1 from text in clipboard, ignoring linebreaks (word_paste.py)
2. Paste up to highlighted text

# DingTag Auto Annotate Workflow
1. Copy your Script before running code (to store into list from clipboard)
2. Run the code and observe controls
   - F8                :: Pastes next word in Script
   - F7                :: Goes back by 1 word (use when you accidentally paste too much)
   - F9                :: SMART SYNC, pastes up to last few words a copied text 
   - ctrl + F9         :: Select all (ctrl+a), Copy (ctrl+c) and SMART SYNC (F9)
   - ctrl + shift + F9 :: Clicks 2 points on screen (get to annotation result box), Plays audio before entering annotation box (ctrl+p), runs ctrl+shift+F9, Goes to next task (increment task in url by 1)
   - Numpad decimal    :: Ctrl+shift+right to highlight text while reading script
3. Run fifth control
4. Make amendments if needed to annotation result
5. press "PAUSE" to submit, it will go to next task and run everything again
